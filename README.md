# PlarailTimetable
# プラレール運転会管理システム

プラレール運転会用のダイヤ管理ツールです。

機能

- 駅別時刻表表示
- 発車済み管理
- 遅延表示
- 運休表示
- 列車詳細表示

※本ツールはダイヤグラム生成くんで事前に作成したファイルが必要です。
- 初めにダイヤグラム生成くんで制作したhtmlファイルを「ファイルを選択」から選択して、ロードします。
  
- ロードされたら自動的に全駅表示されるので、見たい駅をクリックします。
  
- 終点以外の駅であれば、(それぞれの終点駅)方面というボタンがあるので、そのボタンをクリックすれば、そちらの方面の時刻表を見ることができます。(A駅方面をクリックすると、A駅方面の時刻表が出る。)
  
- 時刻表の番号をクリックすることで、発車済、遅延、運休のいずれかを選べます。
  
- 三つのうちどれかを選んだら、「設定」をクリックします。そうすると、時刻表の時刻に反映されます。

- また、時刻表の時刻を長押しすると、その列車の発着時刻を見ることができます。
